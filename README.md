# The Creator Clone ­¤¦¼

**The Creator Clone** is an AI-powered SaaS built for YouTube creators. It analyzes the transcript of any existing YouTube video to extract the creator's unique personality and pacing (their "DNA"). Then, using Google's Gemini 2.5 Flash model, it ghostwrites highly-engaging, retention-optimized scripts that sound *exactly* like them.

## ­¤öź Features

- **Instant Tone Extraction:** Paste a YouTube URL. The app automatically fetches the transcript (with robust language fallbacks), analyzes the pacing, vocabulary, and humor, and generates a precise "Personality DNA" profile.
- **Auto-Loading Profiles:** Automatically fetches and loads your saved `DNA` from the database securely via Supabase as soon as you log in.
- **Customizable Generation Settings:**
  - **Tone Intensity Slider:** From "Chill & Laid-back" (1) all the way up to brutally direct "Savage Mode" (5).
  - **Hook Styles:** Choose how the video opens (Bold Claim, Question, Narrative Story, or Shocking Data Drop).
- **Premium Script Editor UI:** 
  - Real-time Markdown rendering.
  - Smart syntax highlighting that automatically grays out `[STAGE DIRECTIONS]` and `[B-ROLL]` callouts so the spoken script stands out.
  - 1-click script export to `.txt`.
- **Mode Switching:** Specifically engineered prompts optimized for **Shorts/Reels** vs. **Long-Form** content.

## ­¤ÜĆ Getting Started

### Prerequisites
You will need API keys for **Clerk**, **Supabase**, and **Google Gemini**. 

### 1. Clone the repository
\`\`\`bash
git clone https://github.com/ZENOX01/creator-clone.git
cd creator-clone
\`\`\`

### 2. Install dependencies
\`\`\`bash
npm install
\`\`\`

### 3. Setup Environment Variables
Create a \`.env.local\` file in the root directory and add the following keys:
\`\`\`env
# Clerk Auth
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
CLERK_SECRET_KEY=your_clerk_secret_key

# Supabase (Credits & Profiles)
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key

# Google Gemini
GEMINI_API_KEY=your_gemini_api_key
\`\`\`

### 4. Supabase Database Schema
You will need a \`user_credits\` table in Supabase to track script generations and save Personality DNA.

\`\`\`sql
create table public.user_credits (
    id bigint generated by default as identity primary key,
    user_id text not null unique,
    credits integer default 5 not null,
    saved_dna text,
    created_at timestamp with time zone default timezone('utc'::text, now()) not null
);
\`\`\`

### 5. Run the Development Server
\`\`\`bash
npm run dev
\`\`\`
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
